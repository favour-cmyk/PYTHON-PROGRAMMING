## 📋 Projects

### 1. Mathematical Calculator (`calculator.py`)

A console-based calculator that performs common arithmetic operations.

**Features:**
- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Exponentiation
- Invalid input handling
- Division-by-zero error handling

**Usage:**
```bash
python calculator.py
```

---

### 2. CGPA Calculator (`cgpa_calculator.py`)

A program that calculates cumulative grade point average (CGPA) and provides degree classification based on student performance.

**Features:**
- Accept course details
- Calculate grade points
- Compute CGPA
- Display degree classification

**Usage:**
```bash
python cgpa_calculator.py
```
