# PYTHON-PROGRAMMING


COS-202 Python Programming Projects

This repository contains Python projects completed for COS 202 – Computer Programming II. The programs were developed to demonstrate the use of Python in solving mathematical and academic-related problems.

Projects

Mathematical Calculator ("calculator.py")

A console-based calculator that performs common arithmetic operations, including addition, subtraction, multiplication, division, modulus, and exponentiation. It also handles invalid inputs and division-by-zero errors.

CGPA Calculator ("cgpa_calculator.py")

A program that accepts course details, calculates grade points and CGPA, and displays the appropriate degree classification based on the student's performance.

Concepts Used

- Functions
- Loops
- Conditional statements
- User input and output
- Error handling
- Basic arithmetic operations

Running the Programs

python calculator.py

python cgpa_calculator.py

Summary

These projects showcase fundamental Python programming concepts and demonstrate how simple applications can be used to solve real-world computational tasks.
